# Лабораторная работа: контейнеризация и CI

## Что делает программа

Это простое веб-приложение на языке Python с использованием фреймворка Flask.  
Приложение запускает веб-сервер, который обрабатывает HTTP-запросы.

При переходе на главную страницу сервер возвращает текстовое сообщение:  
**"Здравствуйте Андрей александрович. Выполняю лабораторную работу Леонтьев Дмитрий Цтс_32"**

## Запуск проекта и используемый порт

Программа работает 5000 порту.

### Способ 1 — локальный запуск (без Docker)

```bash
pip install -r requirements.txt
python app.py
```

### Способ 2 — запуск в контейнере Docker
```bash
docker build -t lab_app .
docker run -p 5000:5000 lab_app
```
